选择用于此质量阀的代码覆盖率计算的基线。 支持以下不同的基线：
整个项目 - PROJECT
整个项目的覆盖率。这是一个绝对值，在不同的构建之间可能不会有太大的变化。

整个项目(与参考构建的差异) - PROJECT_DELTA
当前构建和参考构建的项目覆盖率之间的差异。团队可以使用这个增量值来确保覆盖率不会降低。

修改的代码行 - MODIFIED_LINES
修改行的覆盖率(例如，在拉取或合并请求的修改行中)将只会关注新的或修改过的代码。

修改的代码行(与整个项目的差异) - MODIFIED_LINES_DELTA
当前构建的项目覆盖率和修改行覆盖率之间的差异。团队可以使用这个增量值来确保拉取请求的覆盖率优于整个项目的覆盖率。

修改的文件 - MODIFIED_FILES
修改文件的覆盖率(例如，在拉取或合并请求所关联的文件中)将只会关注新的或修改过的代码。

修改的文件(与整个项目的差异) - MODIFIED_FILES_DELTA
当前构建的项目覆盖率和修改文件覆盖率之间的差异。团队可以使用这个增量值来确保拉取请求的覆盖率优于整个项目的覆盖率。
